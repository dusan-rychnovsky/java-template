# Java Template

An empty template for new Java projects.

Sets up:
 * JAR with dependencies,
 * Log4J,
 * JUnit and Mockito,
 * Github Action for releases.

After checking out the template, you need to replace:
 * [ARTIFACT-ID],
 * [PROJECT-NAME],
 * [GITHUB-URL], and
 * [MAIN-CLASS]

in
 * pom.xml, and
 * .github/workflows/create-release.yml.
